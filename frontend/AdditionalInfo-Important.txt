Обяснение на конфигурациите:
package.json:
	1	Задължителни зависимости:
	◦	Shopify специфични пакети (app-bridge, polaris)
	◦	React и свързани библиотеки
	◦	axios за HTTP заявки
	2	Dev зависимости:
	◦	Vite с React plugin
	◦	ESLint с Shopify конфигурация
	◦	Prettier за код форматиране
	◦	Vitest за тестове
	3	Скриптове:
	◦	dev: Стартира development сървър
	◦	build: Създава production build
	◦	lint: Проверява кодови стилове
	◦	test: Пуска тестовете
vite.config.js:
	1	React поддръжка:
	◦	Официален Vite React plugin
	◦	JSX автоматично преобразуване
	2	Aliases:
	◦	Създава псевдоними за по-лесни импорти
	◦	@ сочи към src директорията
	3	Development сървър:
	◦	Фксиран порт 3000
	◦	HMR конфигурация
	4	Production build:
	◦	Код splitting за vendor и shopify пакети
	◦	Оптимизирани изходни файлове
	5	Тестова конфигурация:
	◦	JSDOM среда за тестове
	◦	Покритие на код (coverage)
Допълнителни препоръки:
	1	Добавете .npmrc файл за точно версииране:
text

engine-strict=true
save-exact=true
	2	Създайте setupTests.js за тестове:
javascript

import '@testing-library/jest-dom/vitest';
import { installGlobals } from '@remix-run/node';
installGlobals();
	3	Добавете .eslintrc.json за кодови стилове:
json

{
  "extends": ["@shopify/eslint-plugin/react"],
  "rules": {
    "react/react-in-jsx-scope": "off"
  }
}
Тези конфигурации ще:
	•	Осигурят бърз development workflow
	•	Поддържат модерни React практики
	•	Осигурят добра производителност
	•	Дадат възможност за тестване
	•	Спазват Shopify стандарти за код
